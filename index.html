    PROJECTS

    Recommendation Systems based on a Knowledge graph

    Spring 2022
    Research project in NWAFU.
    • Machine Learning
    • Tensorflow

    As the main technique used in the recommendation system, Collaborative Filtering usually suffers from limited performance due to the high sparsity of user-item interactions. A Knowledge Graph (KG), thus, is introduced to mitigate the problem above. Among the already developed methods in the area of KG, a useful one is called RippleNet. It assumes a user is interested in the candidate items that have strong relevance to her historical items. Thus historical items with strong relevance to candidate items are used to embed the user. However, each of the historical items with strong relevance to the candidate items has the same contribution of embedding the user, resulting in unsatisfactory performance in the top-k recommendation task. I proposes a KG-based model Rec-RN, an end-to-end framework, for recommendation systems. This method designs weight based on user preference with controlling each historical item's contribution to user embedding. And Rec-RN computes the weight with corresponding items to form the final high-quality user embedding. Finally, Rec-RN concatenates the user embedding with the candidate item embedding as the input of Multi-layer Perception (MLP) for predicting the clicking probability.
